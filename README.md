# .NET Core Web API - CRUD Mastery Project

Bu proje, .NET Core Web API kullanarak temel CRUD (Create, Read, Update, Delete) operasyonlarını ve RESTful mimariyi içselleştirmek amacıyla geliştirilmiştir.

## 🚀 Gelişim Süreci ve Metodoloji

Bu repo, "kas hafızası" oluşturma odaklı bir öğrenme sürecini yansıtmaktadır:

1.  **Aşama (Iteration 1):** Temel Controller yapısı, Dependency Injection ve List tabanlı veri yönetimi kurgulandı.
2.  **Aşama (Iteration 2):** Kod bakmadan tekrar yazılarak mimari yapı (Routing, HTTP Verbs, Status Codes) pekiştirildi.
3.  **Gelecek Aşama:** InMemory list yapısından **Entity Framework Core** kullanılarak gerçek bir veritabanı (SQL Server/PostgreSQL) mimarisine geçiş yapılacaktır.

## 🛠 Kullanılan Teknolojiler

* **.NET 8.0 / 9.0** (Web API)
* **ASP.NET Core Controllers**
* **Swagger / OpenAPI** (API Test ve Dokümantasyon)

## 📌 Öğrenilen Kazanımlar

* REST ilkelerine uygun endpoint tasarımı.
* `IActionResult` ile doğru HTTP durum kodlarının dönülmesi.
* Postman/Swagger üzerinden API test süreçleri.
* Modelleme ve veri transferi mantığı.
